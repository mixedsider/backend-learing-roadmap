# Chapter 07. API 문서화 (Swagger)

## 실행 방법

```bash
npm install
npm run dev
```

## API 문서 확인

서버 실행 후 아래 주소에서 Swagger UI를 확인합니다.

```text
http://localhost:3000/api-docs
```

## 학습 포인트

- `swagger-jsdoc`으로 JSDoc 주석을 OpenAPI 명세로 변환
- `swagger-ui-express`로 API 문서 화면 제공
- 요청 body, 응답 schema, 인증 방식 문서화
- API 구현 코드와 문서의 위치를 함께 관리하기
